# Car Marketplace App 

Навчальний проєкт для перегляду та пошуку автомобілів з використанням React та DummyJSON API.

## 🔗 Посилання на проєкт
[Живий демо-перегляд](https://car-showroom-62xz.vercel.app)

## 🛠 Функціонал
- **Головна сторінка**: Список автомобілів, завантажених з API.
- **Пошук**: Динамічна фільтрація за назвою та описом (через URL-параметри).
- **Детальна сторінка**: Перегляд повної інформації про авто за ID.
- **Кастомний слайдер**: Реалізовано перегляд галереї зображень без сторонніх бібліотек.
- **Відгуки та характеристики**: Вивід технічних даних, рейтингу та коментарів користувачів.
- **Адаптивний дизайн**: Коректне відображення на мобільних пристроях.
